ПЕРВОНАЧАЛЬНАЯ НАСТРОЙКА

    git config - позволяет просматривать и устанавливать параметры, контролирующие все аспекты работы Git'а.
        
        Параметры могут быть сохранены в трёх местах:
            --system - определяет общие параметры для всех пользователей системы и для всех их репозиториев.
                Эти параметры сохраняются и читаются из файла /etc/gitconfig (для Windows, в папке Git\mingw64/
                /\etc\gitconfig)
            --global - определяет параметры конкретного пользователя.
                Эти параметры сохраняются и читаются из файла ~/.gitconfig (для Windows, в папке пользователя)
            --local - определяет параметры конкретного репозитория.
                Эти параметры сохраняются и читаются из файла .git/gitconfig
        
        Настройки на каждом следующем уровне подменяют настройки из предыдущих уровней
        Настройки пользователя*
            git config --global user.name "John Doe"
            git config --global user.email johndoe@example.com
        Выбор редактора
            git config --global core.editor emacs
        Утилита сравнения
            git config --global merge.tool vimdiff
        Проверка настроек
            git config --list
            git config parametr
