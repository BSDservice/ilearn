ОТМЕНА ИЗМЕНЕНИЙ
Не все операции отмены в свою очередь можно отменить!


git commit --amend - эту команду можно вызвать для изменения последнего коммита, предварительно проиндексировав изменения.
 	Или просто изменить комментарий к коммиту, для этого индексация не требуется.

git reset HEAD <file> - удаляет индексацию изменений файла из подготовительной области.

git checkout -- <file> - возвращает состаяние файла к последнему коммиту.
